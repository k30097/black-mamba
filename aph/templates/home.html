- Welcome to the Apache Persuasion Hold API -

This RESTful API is a tool that aims to speed up the reconnaissance step of the penetration testing process.
It includes an Nmap tool, a Dirbuster and an HTML scanner that identifies vulnerable tags using BeautifulSoup.
It was developed by Khadim Kante between November 2023 and March 2024 as the final project for his Cyber Security and Network degree from Glasgow Caledonian University.
The supervisor was professor David Hendry.
