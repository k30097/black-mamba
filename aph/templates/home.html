- Welcome to the Security Report Generator -

This is a RESTful API that can be used as a pen testing tool and security report generator. It was developed by Khadim Rassoul Kante between November 2023 and INSERT-MONTH 2024 as final project for his Cyber Security and Network degree from Glasgow Caledonian University. The supervisor was professor David Hendry.

The app can perform different type of scans or attacks on the selected target. based on the results, the app then passes the information through to the OpenAI language model to generate a security report on the vulnerabilities that were found.
